# Сайт-витрина с ИИ-генератором постов

Проект по благоустройству территорий.

### Что реализовано:
*   **Сайт-витрина:** лендинг для услуг по ландшафтному дизайну.
*   **ИИ-генератор:** создание рекламных постов через API OpenRouter (модель Hermes/Llama).
*   **Автопостинг:** автоматическая публикация готового контента на стену группы ВКонтакте.

### Технологии:
HTML/CSS, Python, Streamlit, VK API, OpenRouter API.
